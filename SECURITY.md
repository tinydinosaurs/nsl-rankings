# Security Policy

## Supported versions

Only the deployed version on `main` is supported. There are no backports.

## Reporting a vulnerability

**Please do not open a public GitHub issue for security problems.**

Use GitHub's [private vulnerability reporting][gh-private] for this repo, or
email the maintainer directly. Include:

- A clear description of the issue
- Steps to reproduce (or a proof-of-concept if you have one)
- The impact, as best as you can assess it
- Any suggested fix

You should expect an acknowledgement within a few days. Once a fix is in
place, we'll coordinate disclosure timing with you.

Thanks for helping keep this project (and its users) safe.

[gh-private]: https://docs.github.com/en/code-security/security-advisories/guidance-on-reporting-and-writing-information-about-vulnerabilities/privately-reporting-a-security-vulnerability
