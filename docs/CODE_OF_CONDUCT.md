# Code of Conduct

This project follows the [Contributor Covenant 2.1][cc].

In short: be kind, be patient, assume good faith, and remember that the people
on the other end of every PR and issue are humans giving their time. Harassment,
discrimination, and personal attacks aren't tolerated.

To report a violation, contact the project maintainer privately (see
[SECURITY.md](SECURITY.md) for contact info). Reports are handled
confidentially.

[cc]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
